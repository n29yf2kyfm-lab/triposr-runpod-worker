# Hunyuan3D-2.1 worker

Image -> 3D (shape + PBR texture) serverless worker, same I/O contract as
`trellis/`. VRAM: ~10GB shape-only, ~29GB full -> deploy on a 48GB pool
(A40/A6000/L40S) with `texture=true`, or 24GB with `texture=false`.
